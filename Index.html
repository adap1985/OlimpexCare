Conectar con Google Sheets

Pega aquí la URL de tu Apps Script Web App para activar la sincronización multiusuario.

La URL se guarda en este navegador. Cada usuario debe pegarla una vez.

Pasos para configurar
  1. Abre tu Google Sheets en tu cuenta @olimpex.com.
  2. Ve a Extensiones → Apps Script.
  3. Borra el contenido existente y pega todo el código del archivo Code.gs que te entregamos.
  4. Haz clic en Guardar (ícono de disco) y luego ejecuta la función initSheets para crear las hojas automáticamente.
  5. Ve a Implementar → Nueva implementación. Tipo: Aplicación web. Ejecutar como: Yo. Acceso: Cualquier usuario. Haz clic en Implementar.
  6. Autoriza los permisos cuando te lo pida Google.
  7. Copia la URL de la aplicación web que aparece y pégala arriba.
  8. Comparte esa misma URL con cada persona del equipo para que la peguen en su navegador.
